Exercise 1


What is a bit?
A bit is the smallest, most fundamental unit of information in computing. It represents a single choice between two mutually exclusive possibilities, usually represented as a 0 or a 1

Exercise 2

Why does adding one more bit double the number of possible values?
Each previous combination (0 and 1) now has two versions, one beginning with 0 and one beginning with 1.

Therefore, every additional bit doubles the total number of possible combinations, giving 2ⁿ combinations for n bits.

Exercise 4

If one byte can store 256 different values, why is that useful for storing text?
A capacity of 256 values is highly useful for text because it is large enough to assign a unique number code to every single character on a standard English keyboard.

Exercise 5 (Reasoning)

Suppose someone says:

"A bit stores the number 1."

Why is that statement incorrect?
This statement is incorrect because it only describes half of what a bit can do, and confuses a state with a permanent value.A bit is a container, not a fixed number. It can store a 0 or a 1
