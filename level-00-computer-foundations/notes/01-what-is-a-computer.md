# What Is a Computer?
 it is programming machine that takes an data and execute instruction and producea result.

## Intuition

Imagine I lock you in a room with:

a notebook,
a pencil,
a set of instructions.

I tell you:

If someone gives you two numbers, add them.

If someone gives you a sentence, count the words.

If someone gives you a list of names, sort them alphabetically.

You are now acting like a computer.

You:

receive information,
follow instructions,
produce results.

A computer is fundamentally the same thing, except it does this:

billions of times per second,
with electricity,
following extremely simple instructions.
